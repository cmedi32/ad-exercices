# Technology Stack

## Core Technologies
- **Language:** Java 19
- **Build Tool:** Apache Maven
- **UI Framework:** OpenJFX (for GUI applications)

## Libraries & Frameworks
- **Logging:** Logback (configured via `logback.xml`)
- **Testing:**
  - JUnit 5 (Jupiter)
  - AssertJ
- **Code Quality:**
  - Checkstyle
  - PMD

## Distribution
- Docker (via `Dockerfile`)
- GraalVM (mentioned in `pom.xml`)
